---------------RESPUESTAS TEÓRICAS----------------
¿Para qué se utilizan los operadores lógicos en programación?
Los operadores lógicos en la programación se utilizan para evaluar si se sumple una condición o no dependiendo que operador sea el que estemos utilizando, en pocas pálabras evalua si se 
cumplen las condiciones permitiendo que el programa tome decisiones y llegue a una conclusión si es falso o verdadero.

¿Por qué es importante declarar correctamente el tipo de dato de una variable?
Es importante porque las variables le indican al programa que tipo de dato deben de guardar como por ejemplo int, String, float entre otras. Las variables hacen que el programa que
estemos realizando no tenga errores y un dato importante es que las variables se almacenan en memoria dependiendo que tipo sea y si colocamos mal una variable esto nos dará un error
y no se ejecutara bien el programa.
